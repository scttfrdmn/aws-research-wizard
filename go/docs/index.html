🚀 Get Started in 5 Minutes

1

Download & Install

Download the single binary and install system-wide

2

Configure AWS

Setup AWS credentials and test connectivity

3

Create Workflow

Define declarative YAML workflows for your data management

4

Monitor & Optimize

Track progress, costs, and performance with real-time dashboards

# Download and install
wget https://github.com/aws-research-wizard/releases/latest/aws-research-wizard-linux-amd64.tar.gz
tar -xzf aws-research-wizard-linux-amd64.tar.gz
sudo mv aws-research-wizard /usr/local/bin/

# Configure AWS credentials
aws configure

# Analyze your research data
aws-research-wizard data analyze /data/genomics --domain genomics

# Get intelligent transfer recommendations
aws-research-wizard data recommend --source /data/genomics --destination s3://my-bucket/

# Execute optimized workflow
aws-research-wizard data workflow run genomics-project.yaml

# Monitor transfer progress
aws-research-wizard monitor --dashboard
